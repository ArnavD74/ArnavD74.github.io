COVID Atlas

Built By a Global Community

COVID Atlas provides easy to understand local-level data – including local charts, maps, and insights – related to the COVID-19 pandemic. This project is community-developed, open source, and publishes all data, research, and related findings in the public domain.

I assisted in creating a Python webscraper that collected COVID-19 statistics data from the Indian Ministry of Health. This data increased the granularity from a country-level to state-level in the COVID Atlas database, as well as a source in the coronadatascraper.
